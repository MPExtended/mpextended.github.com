Download WebMediaPortal IIS Edition

The current version of MPExtended WebMediaPortal IIS Edition is 0.4.3, released on 27 March 2012. Click here to download it. This is a version of WebMediaPortal for the IIS webserver. You have to manually configure IIS with this edition. If you don't have IIS or want an easy setup, use the standalone version.

Source

Our source is hosted in Git on GitHub. See the repository page for more information, where you can also download the latest development version in zip or tar format. If you want to contribute, the project can be cloned with Git from git://github.com/MPExtended/MPExtended.

You can find the last stable version 0.4.3 in the repository under the name v0.4.3, or download it directly in zip or tar format.
