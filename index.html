MPExtended

MPExtended is a project that aims at providing access to MediaPortal from all over the world, using clients for your webbrowser, Android, iOS, and more.

MPExtended is a project that aims at providing access to MediaPortal from all over the world. To reach this goal, multiple applications are in development:

  • WCF Services for both the client and TV Server, to open up the databases for all applications
  • A streaming service with on-the-fly transcoding to allow watching media on computers, tablets and smartphones
  • WebMediaPortal, a webinterface for MediaPortal
  • Other people are also developing clients based upon our services for Android, iOS and MP2.

Support

Documentation for MPExtended is available on the MediaPortal wiki. You can get support in the MediaPortal forums. The developers are also available on IRC, in #mp-extended on irc.freenode.net. Please provider your logs and configuration if you have any problems.

Download

Currently no stable release is available. You can download the latest development version in zip or tar format from Github. If you want to contribute, you can also clone the project with Git from git://github.com/MPExtended/MPExtended.
